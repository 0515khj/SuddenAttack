## 🚀 프로젝트 주요 기능  
1. **검색**: 제품명으로 빠르게 검색 가능  
2. **좋아요**: 클랜 페이지에서 좋아요 버튼 구현  
3. **정렬**: 제품 정렬 기능 (이름순, 가격순, 카테고리)  
4. **수량 관리**: 장바구니에서 수량 증가/감소  
5. **장바구니**: 상품 추가/삭제 및 장바구니 비우기
